# Wifi_Strength_Mapping
A Flask-based web app that measures WiFi signal strength across different rooms, stores results, visualizes them with charts, and provides full CRUD controls. Built using Python, Flask, Chart.js, and JSON.
My app wifi_strength_mapping also petted as "range_undo_monee"complete WiFi Strength Mapping Web Application built using Python, Flask, HTML, CSS, and JavaScript.
The app measures WiFi signal strength per room, stores the data in a JSON file, and renders a beautiful visual dashboard including colored strength indicators and a bar graph using Chart.js.
Supports full CRUD (create retrieve update delete)operations (add, update, delete, reset) and provides a clean, mobile-friendly UI.
